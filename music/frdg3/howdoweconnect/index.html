In November 2024, before making my remix of Andiamo, I came across Michael e's track "True Love" and played around with it a bit. This song is a refined version of that, with some added elements that make me consider it a standalone, worthy of being released without it being a "remix".

The version on this site (via the preview and download links) has different mastering than the one on streaming services, so I've titled it "how do we connect? (web version)" in the file metadata. I won't mention if I do this in the future, but I'll put "web version" in the file name for the sake of clarity if this ends up being the case again.

The album art was made by someone who doesn't want to be credited. We'll call them "Mike".
