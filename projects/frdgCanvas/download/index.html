frdgCanvas

Downloads

frdgCanvas is built and tested for Windows and Linux on x86 architecture. However, it should work on any system that supports GTK and Python!

Run it from the script or find out how to compile it for your system here.

Windows

Download installer (.exe)
You can access the custom brushes directory at %appdata%/frdgCanvas/brushes.

Linux

Download executable
You can access the custom brushes directory at ~/.config/frdgCanvas/brushes.

My Custom Brushes

Download .zip archive
I've made some of my own custom brushes, download and extract this archive, then install the .frdgbrush files within the software.
