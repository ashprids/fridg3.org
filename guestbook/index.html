commands

utility and customization

/report <username> <reason>
Example: "/report walnuts26 being rude"
Report a specific user for rule violation

/ignore <username>
Example: "/ignore walnuts26"
Hide messages from a specific user for 15 days

/unignore <username>
Example: "/unignore walnuts26"
Stop hiding messages from a specific user

/color [red/orange/yellow/green/teal/blue/purple/pink]
Example: "/color orange"
Change your username's text colour for everyone

/me <text>
Example: "/me is feeling happy!"
Display a status message alongside your name

/roll <number>
Example: "/roll 20"
Roll a virtual die with the specified number of sides

/coin
Flip a coin

/time
Display the current time (according to the server)

/ping
Pong!

/unreserve
Clear your current username reservation, allowing you to choose another and for others to choose yours

Some commands have aliases (e.g. /ignore = /block, and /color = /colour).

Use /help or /? to view this help page.

rules

terms and guidelines

  1. No spamming or flooding the guestbook with messages
  2. No offensive, abusive, or inappropriate language
  3. No personal attacks or harassment of other users
  4. Respect others' privacy; do not share personal information
  5. The use of bots or automated scripts to post messages is prohibited
  6. One username per IP; no "name-sniping" or trying to claim multiple usernames
  7. Ensure all messages comply with British and local law
If you see a violation of these rules, please use the /report command and the incident will be reviewed.

In the meantime, you can use /ignore to hide a user's messages.
