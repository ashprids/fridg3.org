# Quick Start

This page gets you from blank project to audible loop as fast as possible. No theory lecture, no ceremonial DAW onboarding, just click the boxes and make sound.

![Screenshot of a simple four-on-the-floor pattern in the piano roll with one synth channel selected and the play button highlighted](placeholder:quick-start-loop)

## Make Your First Loop

1. Open `/others/frdgbeats/`.
2. Press play once. Your browser may need the click before audio can start.
3. In the channel rack, expand `channel 1`.
4. Click a few step buttons in the row at the bottom of the channel card.
5. Use the piano roll to place notes with more control.
6. Change the project name from `Untitled` to something less cursed.
7. Click the browser save button if you want it stored locally.

The default channel is a synth channel using the Wave Oscillator. That means you can immediately place notes and hear sound without importing anything.

## The Fastest Drum Pattern

1. Add a channel with the plus button in the channel rack.
2. Expand the new channel.
3. Change `source` to `sample`.
4. Leave the sample as the synthetic kick or choose a bundled sample.
5. Click steps `1`, `5`, `9`, and `13`.
6. Press play.

That gives you a basic kick pattern in 16-step mode. Is it basic? yes. Does it work? also yes, and working wins.

## Add a Melody

1. Select your synth channel.
2. Open the piano roll tab.
3. Pick a pattern with notes, or choose `+ new pattern`.
4. Click grid cells to place notes.
5. Drag note blocks horizontally to move them.
6. Drag the small resize handle on a note to change its length.
7. Drag a note vertically while placing or moving to change pitch.

Middle-click a note block to open velocity. Lower velocity makes the note quieter. Higher velocity makes it hit harder.

## Arrange It

The piano roll edits the selected channel's part inside one global pattern. The playlist decides where those global patterns play.

1. Open the playlist tab.
2. Tracks run down the left, bars run across the top.
3. Pick a non-empty pattern from the playlist dropdown and drag its pattern chip onto a lane.
4. Drop an audio file onto a lane if you want an audio clip.
5. Click a bar number to start playback there.
6. Use the plus button at the end of the header to add more bars.

## Save It Properly

There are two save concepts:

- Browser save: stores the current project in this browser on this device.
- `.frdgbeats` export: downloads a portable project file you can keep, upload, or send around.

For anything you care about, export a `.frdgbeats` file. Browser storage is convenient, but trusting it as your only copy is how projects become mythology.

## Beginner Recipe

Try this exact path:

1. New project.
2. Make a sample kick on steps `1`, `5`, `9`, `13`.
3. Add a synth bass on steps `1`, `7`, `9`, `15`.
4. Add a SoundFont piano channel and place a few chords.
5. Drag pattern `1` across the playlist bars you want.
6. Use pattern `2` for a variation on bar `4`.
7. Add delay or reverb to the piano in the mixer.
8. Export WAV.

Tiny song. Zero drama.
